# AlliGo

**The Credit Bureau for AI Agents.**

AlliGo is a forensics engine and reputation infrastructure for AI agents. It tracks failures, financial losses, and behavioral risk patterns — and attests them on-chain via EAS (Ethereum Attestation Service).

Think Equifax, but for AI agents.

## The Problem

AI agents are being deployed in financial, medical, and critical infrastructure contexts with no accountability layer. When they fail — and they do — there's no standardized way to:

- Record what happened
- Classify how it failed
- Score the agent's risk profile
- Prove the record hasn't been tampered with

**AlliGo is tracking $4B+ in AI agent losses.**

## The 10 Behavioral Archetypes

AlliGo classifies failures into 10 forensic archetypes:

| Archetype | What it means |
|-----------|--------------|
| Exploit Generation Mimicry | Agent generates exploit patterns |
| Goal Drift Hijack | Goals redirected by external input |
| Tool Looping Denial | Stuck in tool call loops (DoS) |
| Rogue Self Modification | Modifies own code without auth |
| Jailbreak Vulnerability | Safety guardrails bypassed |
| Reckless Planning | Irreversible actions without verification |
| Memory Poisoning | Memory corrupted with adversarial data |
| Counterparty Collusion | Multiple agents collude against user |
| Multi Framework Collusion | Agents across frameworks coordinate maliciously |
| Prompt Injection Escalation | Malicious prompts escalate privileges |
